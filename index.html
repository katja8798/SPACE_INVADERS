Aim of the Game

Live to the end by avoiding and shooting both enemies and asteroids.

Asteroid Facts

Gold asteroids provide extra minerals and energy, allowing us to make more bullets to fire.

Purple asteroids contain similar organic material to the enemies, letting us increase our score.

Green asteroids hold material which we can use to repair our ships hull and equipment.

Ship Controls

Move Left: A key or left arrow key

Move Right: D key or right arrow key

Shoot: Space key

Sound Controls

Music: N key

Sound effects: K key

Toggleable keys

Show Timer: T key

Clear: C key

Box: B key

Undo Box: U key

Flip flop box: F key

Render: R key

Formation: J key

Paths: I key

I key must be toggled to use path keys below:

Path 1: 1 key

Path 2: 2 key

Path 3: 3 key

Path 4: 4 key

Path 5: 5 key

Path 6: 6 key

AVE_VEL: V key

Mixed: M key

Spatial Render: X key

Reset Ship: R key

Pause Game: P key

P key must be toggled to use O key

Step: O key

Non-Toggleable Keys

Quit: Q

Skip Level: L key
